## Project Overview

This project investigates why Global Superstore achieved continuous sales growth while profitability failed to improve between 2014 and 2017.

Using SQL, Python, Tableau and business analysis techniques, the project identifies the major factors behind declining profit margins and provides actionable business recommendations.

---

## Business Questions

This project answers the following questions:

- How did sales and profit evolve over time?
- Was profitability deterioration caused by increasing discounts?
- Which product categories generated the largest losses?
- Which markets suffered the most under aggressive discounting?
- Did high-discount orders also increase return risk?
- What business actions should management take?
